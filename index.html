How It Works — Priority Encoding Explained
The clever part of this design

Imagine a hospital triage system: if a critical patient AND a mild patient both arrive, the doctor goes to the critical one first. This circuit does the same thing. If Zone 1 (master bedroom) AND Zone 3 (motion sensor) both trigger simultaneously, only Zone 1's LED lights — Zone 1 wins.

Real-world use: Zone 1 = Master bedroom door (highest risk). Zone 2 = Living room window. Zone 3 = Motion sensor in hallway (lowest — could be a pet). If the burglar smashes the bedroom door AND triggers the hallway sensor, you see Zone 1 LED — the most important alert.

The Logic — What Equations Drive This
Don't skip this — understanding = easier debugging

There are 4 outputs. Here is the logic equation for each one and what it means in plain English:

🔴 Alarm Output — Any zone triggered

Alarm = Zone1 OR Zone2 OR Zone3

The alarm LED fires whenever ANY zone is active. Simple OR gate chain.

🔴 Zone 1 LED — Highest priority

Zone1_LED = Zone1

Zone 1 always wins. No masking needed — it lights up whenever Zone 1 is active, regardless of what else is happening. Direct wire from Zone 1 sensor to its LED.

🟠 Zone 2 LED — Only if Zone 1 is NOT active

Zone2_LED = Zone2 AND (NOT Zone1)

Zone 2's LED only lights if Zone 2 is triggered AND Zone 1 is NOT triggered. If Zone 1 is ON, Zone 1 steals the display. This needs: 1× NOT gate + 1× AND gate.

🟡 Zone 3 LED — Only if BOTH higher zones are NOT active

Zone3_LED = Zone3 AND (NOT Zone1) AND (NOT Zone2)

Zone 3 only displays if it's alone. If Zone 1 or Zone 2 is also active, Zone 3 stays silent. This needs: 2× NOT gates + 2× AND gates.

Know All 3 ICs — Pinouts
All three chips follow the same power pin layout: Pin 14 = VCC, Pin 7 = GND
⚠️ All three chips (74HC08, 74HC32, 74HC04) use the same DIP-14 package. The only difference is what happens inside. Always read the label!
Memory trick:
74HC08 = AND (8 looks like two circles joined — AND gate!)
74HC32 = OR (32 gates... just remember OR comes AFTER AND alphabetically)
74HC04 = NOT (04 = zero/none = NOT nothing = inverter!)

Build Steps — Work in Order
Build one section at a time and test before moving on
1
Place All 3 ICs on the Breadboard
Space the three ICs at least 5 rows apart so you have room to wire between them. All three straddle the centre gap. Notch facing left on all three.
💡 Label a sticky note next to each: "AND", "OR", "NOT" so you never mix them up while wiring.
2

Each of the 3 ICs must have its own power and ground connections. Do not skip any IC.

4
Build the Alarm OR Gate (Alarm LED)
The alarm LED should fire if ANY zone is active. Chain two 2-input OR gates:

• Gate 1 (74HC32): Input A = Zone1, Input B = Zone2. Output = (Z1 OR Z2)
• Gate 2 (74HC32): Input A = Gate1 output, Input B = Zone3. Output = Alarm
• Connect Alarm output → 330Ω → Red LED (alarm) → GND
⚠️ The 74HC32 has 4 gates. You're using gates 1 and 2 here. Pins: Gate1 = 1,2,3. Gate2 = 4,5,6.
5
Wire Zone 1 Priority LED (Direct)
No logic needed here. Zone 1 always has highest priority and simply connects directly:

Zone1 signal → 330Ω resistor → Red Priority LED (Z1) → GND
💡 Test this step alone first. Zone 1 switch ON → Red priority LED ON. Works? Good — move on.
6
Wire Zone 2 Priority LED (AND + NOT)
Zone 2 only shows if Zone 1 is NOT active:

1. Zone1 signal → NOT gate (74HC04 Pin 1). Output (NOT Z1) comes out Pin 2.
2. NOT Z1 output and Zone2 signal → AND gate (74HC08 Pin 4, Pin 5). Output at Pin 6.
3. Pin 6 → 330Ω → Orange Priority LED (Z2) → GND
7
Wire Zone 3 Priority LED (AND + 2× NOT)
Zone 3 only shows if both Zone 1 AND Zone 2 are NOT active:

1. Zone1 → NOT gate (74HC04 Pin 3→4). Output = NOT Z1
2. Zone2 → NOT gate (74HC04 Pin 5→6). Output = NOT Z2
3. NOT Z1 and NOT Z2 → AND gate 1 (74HC08, Gate 1). Output = NOT Z1 AND NOT Z2
4. That AND output + Zone3 signal → AND gate 2 (74HC08, Gate 2). Output = Zone3 LED signal
5. Output → 330Ω → Yellow Priority LED (Z3) → GND
⚠️ The 74HC04 has 6 NOT gates. Each is: Pin A (IN) and Pin B (OUT) pairs: 1→2, 3→4, 5→6, 9→8, 11→10, 13→12.

Final Pre-Power Checklist
Tick every single box before connecting power
  • 74HC08 (AND) — inserted notch-left, Pin 14 → +5V, Pin 7 → GND
  • 74HC32 (OR) — inserted notch-left, Pin 14 → +5V, Pin 7 → GND
  • 74HC04 (NOT) — inserted notch-left, Pin 14 → +5V, Pin 7 → GND
  • Switch 1 (Zone 1): +5V → SW → junction, 10kΩ from junction to GND
  • Switch 2 (Zone 2): +5V → SW → junction, 10kΩ from junction to GND
  • Switch 3 (Zone 3): +5V → SW → junction, 10kΩ from junction to GND
  • OR gate chain: Zone1 + Zone2 → OR1 → OR2 ← Zone3 → Alarm LED via 330Ω
  • Zone1 signal → 330Ω → Red Priority LED → GND (direct connection)
  • Zone1 → NOT gate → AND gate (with Zone2) → 330Ω → Orange LED → GND
  • Zone1 → NOT gate AND Zone2 → NOT gate → AND → AND (with Zone3) → 330Ω → Yellow LED → GND
  • All LEDs have 330Ω current-limiting resistors
  • Power source is 5V DC — not 9V direct!
  • No wires crossing the centre gap accidentally
